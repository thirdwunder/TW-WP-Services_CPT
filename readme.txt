== Description ==

This plugin creates a Services Custom Post Type with options for categories and tags taxonomies.

== Installation ==

Installing "TW_Services_Plugin" can be done by using the following steps:

1. Download the plugin via github https://github.com/thirdwunder/TW-WP-Services_CPT
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
